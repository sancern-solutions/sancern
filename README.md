# Sancern Solutions Website V2

This is a GitHub Pages-ready static site for **Sancern Solutions** with a dedicated **Smart Cuts** product page.

## Included
- Premium homepage design
- Smart Cuts product page
- Mobile nav support
- SEO meta tags and JSON-LD
- Robots + sitemap
- Lemon Squeezy-ready checkout button hooks
- SVG logo, favicon, cover art, and illustration assets

## Before publishing
Replace these placeholders:
- `https://YOUR-GITHUB-PAGES-URL/`
- `hello@YOURDOMAIN.com`
- `https://YOUR-LEMONSQUEEZY-STARTER-LINK`
- `https://YOUR-LEMONSQUEEZY-PRO-LINK`
- `https://YOUR-LEMONSQUEEZY-STUDIO-LINK`

## GitHub Pages
1. Upload all files to your GitHub repository root.
2. Enable GitHub Pages for the branch you want to publish.
3. Confirm the final site URL.
4. Update canonical URLs, sitemap, robots, and social URLs.

## Lemon Squeezy
The checkout buttons already include the `lemon-link` class and Lemon.js is loaded globally. Replace the placeholder href values with your real checkout links.
